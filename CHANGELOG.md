# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - Unreleased

### Added
- Bookmark counts displayed in category badges
- Dynamic count updates based on search filters
- Visual count badges showing total bookmarks per category
- Count badges that update to show filtered results when searching
- Tag system for bookmarks with tag-based filtering
- Active tag filter display with clear button
- Sorting options: Default (Original Order), Name (A-Z), Name (Z-A), and Category
- Sort preference persistence using localStorage
- Keyboard shortcut: Esc key to clear search input
- Search term highlighting in bookmark names and descriptions

### Fixed
- Fixed search term highlighting functionality - highlights now properly appear in matching text
- Improved highlighting algorithm to correctly match and highlight search terms case-insensitively

### Changed
- Darkened navbar background gradient for better contrast and readability
- Improved category button colors (darker shades) for enhanced text visibility
- Enhanced category count badge styling for better visibility on darker background
- Increased shadow depth on active category buttons
- Improved highlight styling with better visibility (lighter yellow background, shadow effect)
- Disabled browser autocomplete on search input for cleaner user experience

[1.1.0]: https://github.com/ma1c0ntent/bookmarks/releases/tag/v1.1.0

## [1.0.0] - 2025-11-13

### Added
- Initial release of standalone HTML bookmark manager
- Category-based organization system
- Real-time search functionality across bookmark names and descriptions
- Category filtering with visual active state indicators
- "No results found" message when search returns no matches
- Clean, modern UI with gradient backgrounds and smooth animations
- Responsive design that works on desktop and mobile devices
- Comprehensive bookmark collection across Documentation, News, and Training Resources categories

### Fixed
- Fixed typo: "Infrasturcture" → "Infrastructure" in CISA description
- Fixed typo: "Profresionalization" → "Professionalization" in STEPP description
- Fixed typo: "guildlines" → "guidelines" in Keep a Changelog description
- Fixed typo: "communition" → "communication" in Keep a Changelog description
- Fixed typo: "insutry" → "industry" in Defense News description

### Changed
- Improved search functionality to respect category filters
- Enhanced user experience with proper "no results" feedback
